qaio_f(userInput, requests, responses, intentCondition, typecond, intentPatternType, dispout) → {object}

Searches for the highest occurrence of nouns or all words in patterns or pattern strings, with the intent processing option.
Parameters:
Name Type Description
userInput string Input sentence as a string.
requests array Requests as an array.
responses array Responses as an array.
intentCondition number Intent search option: (0) (input -> requests -> reponses), (1) (input -> reponses -> reponses).
typecond number Return type option: (0) Process nouns, (1) Process including stop words.
intentPatternType number Set intent type: (0) for the highest occurrence of input words in the sentence string or (1) for the highest occurrence in the pattern.
dispout boolean (true/false) Display processing output results in terminal.
