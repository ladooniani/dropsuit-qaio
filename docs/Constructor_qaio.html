Class: qaio

qaio(input, intentCondition, typecond, intentPatternType) → {object}

- qaio(null/string, 0-1, 0-1, 0-1) function - Searches for the highest occurrence of nouns or all words in patterns or pattern strings, with the intent processing option. Processes default object instance json key value\n(req_arr: requests), (res_arr: responses) array patterns.

Constructor

new qaio(input, intentCondition, typecond, intentPatternType) → {object}

Parameters:
Name Type Description
input string Input sentence as a string, or keep NULL to process constructor 'input'.
intentCondition number Intent search option: (0) (input -> requests -> reponses), (1) (input -> reponses -> reponses).
typecond number Return type option: (0) Process nouns, (1) Process including stop words.
intentPatternType number Set intent type: (0) for the highest occurrence of input words in the sentence string or (1) for the highest occurrence in the pattern. Note: A number out of range returns a 0 option result.
